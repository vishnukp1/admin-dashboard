# Techon Dashboard

A pixel-perfect, responsive admin dashboard built with React and Vite. Developed as part of a machine task to implement the Figma design.

## Setup
npm install
npm run dev
App runs at: http://localhost:5173

## Tech Stack
React, Vite, Tailwind CSS, Recharts, Lucide React, Google Fonts (Poppins)
